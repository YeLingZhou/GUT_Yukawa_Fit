This folder lists data of observables, including physical quantities used in fitting and predictions in Figure 1 in arXiv:2511.16196, generated by Observables.nb with data in ./Data/Parameters as inputs.
------------
MiNj.dat (i=1,2,3, j=1,2) gives values of model parameters for Model Mi in fitting data Nj in Normal ordering, shown in Fig. 3, 
MiIj.dat (i=  2,3, j=1,2) gives values of model parameters for Model Mi in fitting data Ij in Inverted ordering, shown in Fig. 4.

7 columns are included, {chi^2/n_{obs}, M_1/GeV, M_3/GeV, m_{lightest}/eV, m_{\beta\beta}/eV, \theta_{23}, \delta}
