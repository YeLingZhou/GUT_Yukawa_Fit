This folder lists data of observables, including physical quantities used in fitting and predictions in Figure 1 in arXiv:2511.16196, generated by Observables.nb with data in ./Data/Parameters as inputs.
------------

M1_1st.dat gives values of physical quantities for Model 1 and \theta_{23} in the first octant case, shown in the first line of FIG. 1, 
3 columns are included, each line gives {chi^2/n_{obs}, M_1, M_3}, {chi^2/n_{obs}, m_{lightest}, m_{\beta\beta}}, {chi^2/n_{obs}, \theta_{23}, \delta}


M2_1st.dat gives values of physical quantities for Model 2 and \theta_{23} in the first octant case, shown in the third line of FIG. 1, 
3 columns are included, each line gives {chi^2/n_{obs}, M_1, M_3}, {chi^2/n_{obs}, m_{lightest}, m_{\beta\beta}}, {chi^2/n_{obs}, \theta_{23}, \delta}


M3_1st.dat gives values of physical quantities for Model 3 and \theta_{23} in the first octant case, shown in the fifth line of FIG. 1, 
3 columns are included, each line gives {chi^2/n_{obs}, M_1, M_3}, {chi^2/n_{obs}, m_{lightest}, m_{\beta\beta}}, {chi^2/n_{obs}, \theta_{23}, \delta}


M1_2nd.dat gives values of physical quantities for Model 1 and \theta_{23} in the second octant case, shown in the second line of FIG. 1, 
Data arranged in the same format as M1_1st.dat


M2_2nd.dat gives values of physical quantities for Model 2 and \theta_{23} in the second octant case, shown in the fourth line of FIG. 1, 
Data arranged in the same format as M2_1st.dat


M3_2nd.dat gives values of physical quantities for Model 3 and \theta_{23} in the second octant case, shown in the sixth line of FIG. 1, 
Data arranged in the same format as M3_1st.dat
